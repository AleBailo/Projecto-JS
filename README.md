# AlancenApp

Aplicación creada para el curso de JavaScript de CoderHouse. El objetivo es armar listas colaborativas para realizar compras de supermecado y otros insumos. 


Flow:
1 Login --> si no registro.
2 Dentro de la app --> Ingreso nombre de la lista --> Ingreso nombre del producto, la cantidad y selecciono la categoría --> Agregar producto.
3 Se mostraran listados los productos agregados --> se puede editar la cantidad o borrar el producto 
4 Una vez que tengo todos los productos que quiero agregar --> Crear lista o Borrar todo.
5 Las listas creadas quedaran listadas en la sección "Mis listas" para verlas mas tarde.
